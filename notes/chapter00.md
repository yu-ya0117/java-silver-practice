## 自分なりの理解

### Javaでできること

- コマンドラインアプリケーション
- ウィンドウアプリケーション
- Webアプリケーション
- Androidアプリケーション
- Web API

### Javaの特徴

- 学びやすく標準的な文法である
- 大規模開発を支援するオブジェクト指向や関数型プログラミングに対応している
- 標準命令群や一般公開されているライブラリの種類が豊富
- ゲームから基幹システムまで幅広く使われている

### 実行までの手順

1. Javaソースファイル（.java）を作成
2. javacコマンドでコンパイル
3. クラスファイル（.class）を生成
4. javaコマンドで実行

### Hello Worldプログラム

```
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

### 変数の使い方

変数はデータを保存するための箱。

```
int num = 5;
System.out.println(num);
```

### コンパイル後のクラスファイルの出力先フォルダを指定する方法

```
javac -d フォルダ名 ソースファイル名.java
```

### 別の場所にあるフォルダ（ディレクトリ）をクラスパスとして指定して実行する方法

```
java -cp フォルダのパス 完全修飾クラス名
```
